# Minecraft Plus

Minecraft 26.2 / Fabric 模组。`src/main/resources/data` 会由 Fabric 作为模组内置数据包加载，对所有世界生效。

## 通用附魔与等级突破

- 所有附魔都可以附到任意非空物品上；附魔即使不适用于该物品，也会被保留。
- 不同附魔之间不再互斥，例如“锋利”和“亡灵杀手”可以共存并分别生效。
- `/enchant` 接受的最高等级统一提升到 `2147483647`；铁砧和附魔台仍使用附魔自身的正常等级上限。
- 示例：手持木棍执行 `/enchant @s minecraft:sharpness 2147483647`。

## 爆炸附魔

- 可用于弓、弩和三叉戟，最高 III 级。
- 投射物命中方块或实体时，在命中点生成与等级相同数量、引信为 0 的 TNT。
- 每颗 TNT 消耗玩家背包中的 1 个 TNT；数量不足时本次不触发。
- 发射武器带有“无限”附魔或玩家处于创造模式时不消耗 TNT。
- 获取测试物品：`/give @s bow[enchantments={"mcplus:explosion":3}]`

## 附魔金胡萝卜

- 饥饿值恢复 6，饱和度恢复 14.4，并且饱食时也可以食用。
- 获得伤害吸收 IV（1 分钟）、生命恢复 II（15 秒）、抗火（3 分钟）和抗性提升 I（3 分钟）。
- 合成：中心放胡萝卜，周围 8 格放金块。
- 可在远古城市箱子中以 8% 概率获得。
- 获取测试物品：`/give @s mcplus:enchanted_golden_carrot`

## 旧版附魔金苹果

- 合成：中心放苹果，周围 8 格放金块。
- 获得金块后会解锁配方；成功合成后获得挑战进度“君临天下”。

## TAB 性能显示

- 按住 TAB 时，在玩家列表下方显示彩色的 TPS、FPS、Ping 和内存占用。
- 单人世界直接读取内置服务端 TPS；多人服务器根据服务端时间同步包估算 TPS，进入服务器后约 1 秒显示。

## 构建

需要 Java 25，并在游戏中安装适用于 Minecraft 26.2 的 Fabric API：

```powershell
.\gradlew.bat build
```

产物位于 `build/libs/mcplus-0.1.0.jar`。
